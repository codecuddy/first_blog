First post

January 9, 2015

My name is Charlie Cuddy. I'm currently in my fifth year teaching high school math, mainly second year algebra. I also am the head cross country and head track coach for our boys and girls varsity teams.

I've been using Khan Academy to learn how to write code for several weeks now. I started with the javascript tutorial. There were some pros and some cons to starting in javascript in my opinion. It was way more challenging than html and css and I can see how one might easily get discouraged and want to give up. However, I liked that the projects we built were more interactive which made it more likely that I'd come back and work on the projects.

I've found that using html and css seems to make way more sense. I'm not sure if it is because of the background I had from the tutorials in javascript, or if it is because it really is that much easier. It has been nice that I've recognized some key words and phrases that show up in both. I am looking forward to learning more, and I have even signed up to take an html/css class at my local library next month.

One of my first projects that I carried over from Khan Academy was a basic web page that taught me how to place photos on the webpage while planning the perfect trip.

First Day Using Notepad++

January 13, 2015

I just downloaded notepad++ today and am playing around with it for the first time. I've watched a few youtube videos to gain some pointers. The biggest help those provided was showing me which plugins were helpful to add right off the start. It is nice coming over from Khan Academy where it was expected to memorize a lot and knowing that when I'm using notepad++ that the program really helps you remember. I'm looking forward to playing around and really getting my hands dirty with it. '

Another very helpful tool that Khan Academy showed me yesterday or today was how to use validator.w3.org to validate if my html is ready to roll. I think I'll really learn a lot from having my errors pointed out to me. I am a bit of a perfectionist and look forward to the challenge of correcting my errors so as to avoid them in the future.

My cousin, Patrick, started taking a few courses at UNL this week in computer programming. I hope to be in constant talks with him to see what he is learning and what I can apply. I want to look more into java eclipse and what it offers.
